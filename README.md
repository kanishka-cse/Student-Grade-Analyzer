# Student-Grade-Analyzer
A Python tool to analyze student grades from CSV data.

## Description
Student Grade Analyzer is a Python project that stores student marks in a CSV file, calculates total marks, average, grade, and pass/fail status, and displays students ranked by total marks.

## Features
- Add new student records
- Store data in students.csv
- Calculate Total Marks
- Calculate Average Marks
- Assign Grades (A–F)
- Display Pass/Fail Status
- Rank students based on total marks
- Award 🥇, 🥈 and 🥉 medals to the top three students

## Technologies Used
- Python
- CSV Module

## Project Files
- Grade_Analyzer.py – Main Python program
- students.csv – Stores student records
- README.md – Project documentation

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the program using:

bash
py Grade_Analyzer.py
